# 🌦 Weather AI App

A modern weather web app built with **React.js** that provides real-time weather updates with a clean UI and smart features.

---

## 🚀 Live Demo

🔗 (https://weather-ai-web.netlify.app/)

---

## ✨ Features

* 🌍 Real-time weather data
* 🔍 Search any city
* 📍 Get weather using current location
* 📊 Temperature forecast graph
* 😂 Funny weather advice
* 🌅 Sunrise & Sunset info
* 🌬 Wind, 👀 Visibility, 💧 Humidity
* 🎨 Dynamic weather backgrounds
* 📱 Responsive design

---

## 🛠 Tech Stack

* React.js
* Chart.js
* OpenWeather API
* CSS

---

## 📦 Run Locally

```bash
git clone https://github.com/your-username/weather-ai-app.git
cd weather-ai-app
npm install
npm start
```
